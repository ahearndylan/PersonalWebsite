Pac-Man Game Project

Project Overview

This project involves developing a Pac-Man game using various algorithms like Depth-First Search (DFS) and A* to manipulate game logic. The project serves as both a practical application of algorithm theory and a demonstration of game development skills.

Technologies Used: Python, Pygame, DFS, Dijkstra's, A* Algorithm, Game Development.

GitHub Repository: View the project on GitHub

Project Description

The Pac-Man game project is a software application developed to simulate the classic Pac-Man arcade game. The project explores the implementation of various search algorithms, such as Depth-First Search (DFS), Dijkstra's, and A*, to control the behavior of ghosts and optimize the gameplay experience.

This project not only showcases game development skills but also demonstrates the practical application of complex algorithms in a gaming environment.

Features

The Pac-Man game includes several key features:

  • Classic Pac-Man Gameplay: Recreates the classic gameplay experience with Pac-Man navigating through a maze while avoiding ghosts.
  • Algorithm-Driven AI: Uses Depth-First Search (DFS), Dijkstra and A* algorithms to control PacMan behavior, providing a challenge in AI to beat the game.
  • Customizable Levels: Allows for the creation of custom levels with varying difficulty.
  • High Score Tracking: Tracks and displays the highest scores achieved by players.
  • Responsive Controls: Implements responsive keyboard controls for smooth gameplay.

Setup and Installation

This section provides instructions on how to set up the Pac-Man game locally:

Prerequisites
  • Python 3.x
  • Pygame
Setup Instructions
  1. Clone the repository:
    2. git clone https://github.com/ahearndylan/PacmanGame.git
cd PacmanGame
  2. Install dependencies:
    3. pip install -r requirements.txt
  3. Run the game:
    4. python pacman.py

Project Structure

The project is structured as follows:

  • src/: Contains the source code for the game, including the main game loop and AI logic.
  • assets/: Contains all game assets, such as images and sounds used in the game.
  • levels/: Contains level design files that define the layout and difficulty of each level.
  • requirements.txt: Lists the Python dependencies needed to run the game.

Algorithm Implementation

The Pac-Man game uses the following algorithms to enhance gameplay:

  • Depth-First Search (DFS): Used to generate maze paths and define the initial layout of the game levels.
  • A* Algorithm: Implemented to control ghost movement, making them more intelligent and challenging for the player.
