BudgetApp

Project Overview

This mobile app helps users create custom budgets, track expenses, and discover nearby restaurants and events. It integrates APIs like Yelp and Ticketmaster for personalized, location-based recommendations. Designed for college students and budget-conscious individuals, the app simplifies expense management while promoting responsible spending habits.

Key Features

  • Custom Budgets: Users can create budgets with detailed categories and track expenses in real-time.
  • API Integration: Uses Yelp API for restaurant suggestions and Ticketmaster API for nearby events.
  • Expense Tracking: Real-time tracking of expenditures with a clear breakdown of spending.
  • Progress Visualization: Displays progress bars to show spending compared to set goals.

Technical Details

The app is developed using Kotlin in Android Studio and leverages a Room database to manage budgets, expenditures, and expenses. It uses a ViewModel with LiveData for data persistence and UI updates. The navigation system is implemented via a Nav Graph, allowing seamless transitions between fragments. API requests to Yelp and Ticketmaster provide dynamic, location-based suggestions.

Additional functionality includes:

  • RecyclerView: Used extensively to display budgets, expenses, and suggestions.
  • Fragments: Organized navigation and data presentation using fragment components.
  • MutableLiveData: Ensures smooth updates to the UI based on database changes.
