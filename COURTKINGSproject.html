Court Kings HQ

Project Overview

Court Kings HQ is a fully automated NBA stats platform that delivers daily basketball analytics via Twitter bots and a dynamic, data-driven website. The system tracks nightly leaders across scoring, efficiency, clutch performance, and season stats using Supabase as a central data hub. The website updates in real time and provides a sleek, mobile-friendly experience for fans.

About Court Kings HQ

Court Kings HQ is a personal project designed to engage basketball fans with daily insights using automated bots and live visualizations. The site is powered by Python scripts that process NBA data and push updates to a centralized Supabase database. These updates feed directly into a multi-page HTML site hosted on Render, ensuring fresh and reliable content daily.

Each stat category is managed by a separate Twitter bot that posts leaderboards and syncs data automatically, allowing the platform to scale and adapt over the course of the NBA season.

Project Goals

The Court Kings HQ project set out to:

  • Automate NBA stat tracking and Twitter updates using Python bots.
  • Build a centralized system to store and serve live data with Supabase.
  • Design a sleek, responsive website with individual stat pages and player spotlights.

Project Outcomes

The completed project delivered:

  • A scalable backend powered by Railway and Supabase with automated cron jobs.
  • A multi-page website with real-time data, live Twitter integration, and stat archives.
  • A clean user interface built with HTML, CSS, Bootstrap, and responsive design principles.

Future Plans

Future enhancements for Court Kings HQ include:

  • Expanding bot capabilities to include player comparison and advanced stats.
  • Introducing user interactivity, such as a "Test Bots" feature or live search.
  • Packaging the project for public use or collaboration with NBA content creators.

Technologies Used

Technologies Used: Python, HTML, CSS, JavaScript, Bootstrap, Supabase, Tweepy, JSON, Git, Render, Railway.
