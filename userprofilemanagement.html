Student Registration App

Project Overview

This Android app facilitates user registration and profile management across multiple screens, with seamless navigation and data persistence. The app uses `ViewModel` to retain user data (name, gender, and age) across fragments, enabling users to navigate back to previous screens without losing entered data. Users can switch between Home and Profile screens via a BottomNavigationView for quick access.

Key Features

  • Multi-Screen Navigation: Users can navigate through multiple registration screens with Next and Back buttons.
  • Data Persistence: Uses `ViewModel` to retain user data across fragments, ensuring information persists when navigating between screens.
  • Profile Summary: Displays a summary of user information with an Edit option for modifications.
  • Bottom Navigation: Quick access to Home and Profile screens via BottomNavigationView.
