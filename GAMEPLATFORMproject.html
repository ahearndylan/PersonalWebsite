Game Platform Project

Project Overview

This project involves developing a Game Platform that hosts multiple games, including Plinko, Horse Racing Derby, Roulette, and a Memory Matching Game. The platform is designed to showcase GUI development skills and game logic implementation.

Technologies Used: Java, JavaFX, IntelliJ IDEA, Game Development.

GitHub Repository: View the project on GitHub

Project Description

The Game Platform project is a software application that features multiple games accessible from a single interface. The project explores GUI development using JavaFX and demonstrates the implementation of different game logics, including physics-based ball drop in Plinko, a race simulation in Horse Racing Derby, a color-based roulette system, and a memory matching game with sport icons.

This project not only showcases game development skills but also highlights the use of Java for creating interactive and user-friendly applications.

Features

The Game Platform includes several key features:

  • Multiple Games: Includes Plinko, Horse Racing Derby, Roulette, and Memory Matching, each with its own unique gameplay mechanics.
  • Interactive GUI: Developed using JavaFX, the platform provides an intuitive interface for navigating between games.
  • Scoring System: Tracks and displays the player's performance across different games.
  • Responsive Controls: Implements responsive controls for smooth gameplay across all games.

Setup and Installation

This section provides instructions on how to set up the Game Platform locally:

Prerequisites
  • Java 8 or higher
  • JavaFX SDK
Setup Instructions
  1. Clone the repository:
    2. git clone https://github.com/ahearndylan/GamePlatform.git
cd GamePlatform
  2. Open the project in IntelliJ IDEA: Import the project as a Gradle project.
  3. Run the application:
    4. ./gradlew run

Project Structure

The project is structured as follows:

  • src/: Contains the source code for the game platform, including the main game loop and individual game logic.
  • resources/: Contains all game assets, such as images and sounds used in the games.
  • build.gradle: Contains the build configuration for the project.
  • gradle/: Contains the Gradle wrapper files for easy setup.

The main class structure includes:

  • `GamePlatform.java`: The entry point for the application, managing the main GUI and navigation between games.
  • `PlinkoGame.java`: Manages the logic and UI for the Plinko game.
  • `HorseRacingGame.java`: Manages the logic and UI for the Horse Racing Derby game.
  • `RouletteGame.java`: Manages the logic and UI for the Roulette game.
  • `MemoryMatchGame.java`: Manages the logic and UI for the Matching Game.
