Student Directory App

Project Overview

This Android app loads a list of students from a CSV file and displays them in a scrollable list. Users can view detailed information for each student, including their profile picture, by clicking on a list item. The app is designed to showcase the use of multi-activity navigation, data handling, and RecyclerView in Android.

Key Features

  • Student List: Loads student data from a CSV file and displays it in a RecyclerView.
  • Multi-Activity Navigation: Clicking a student opens a detailed view with additional information.
  • Data Management: The app parses and stores student data in an ArrayList, enabling efficient access and display.
  • Profile Pictures: Displays gender-specific profile images from resources.

Technical Details

The app was built using Kotlin in Android Studio. Student data is loaded from a CSV file stored in the `res/raw` folder and parsed into an `ArrayList`. Each student item is displayed in a RecyclerView using a custom adapter, and clicking an item opens a DetailActivity where additional information is shown.

The `Student` class includes fields for the student's name, gender, email, phone, and address. The RecyclerView uses a custom XML layout for each student entry, displaying the name, phone number, and profile image.
